API pública — Observatorio IA Costa Rica

JSON read-only. CORS abierto. Sin auth ni rate-limit. Fuente: observatorioia.org.

Endpoints

EndpointItemsDescripción
/api/proyectos.json 20 Catálogo de proyectos de IA en el sector público costarricense. Cada entrada incluye institución, estado, fuente oficial y descripción bilingüe ES/EN.
/api/instituciones.json 7 Instituciones públicas con proyectos IA documentados (ministerios, autónomas, judicial, universidades, investigación).
/api/legislacion.json 5 Expedientes de ley relacionados con IA en la Asamblea Legislativa de Costa Rica. Estado actualizado por scraper automatizado.
/api/indicadores.json 3 Indicadores cuantitativos: ILIA 2025 (Índice Latinoamericano de IA), comparativa regional, KPIs hero del observatorio.
/api/brechas.json 7 Análisis de brechas: 7 capacidades que CR no tiene operativas vs Estonia/Singapur (gobernanza IA, X-Road, chatbot ciudadano, etc.).
/api/index.json — Manifest: lista todos los endpoints con metadata.

Formato de respuesta

Cada endpoint devuelve un envelope con metadata:

{
  "version": "0.1.0",
  "lastUpdate": "2026-05-04T...",
  "count": 18,
  "source": "https://observatorioia.org",
  "license": "CC BY 4.0",
  "data": [ ... ]
}

Licencia y atribución

Datos disponibles bajo CC BY 4.0. Si los usás, atribuir "Observatorio IA Costa Rica" (observatorioia.org) con enlace.

Política editorial

Este observatorio es independiente, sin afiliación gubernamental. Cada dato del catálogo tiene fuenteUrl apuntando al documento oficial original. Las actualizaciones se procesan via revisión humana — los scrapers nunca tocan campos editoriales (titulo, descripcion, contexto, lecciones, resumen).
